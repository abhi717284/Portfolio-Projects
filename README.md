# Data Analytics Portfolio Projects

Welcome to my portfolio! This repository showcases a collection of end-to-end data analytics projects, spanning data scraping, cleaning, exploratory data analysis (EDA), and interactive business intelligence dashboards.

## 📁 Repository Structure
* **`/Dashboards`**: Interactive reports and visualizations (Tableau, Power BI, Excel).
* **`/Scripts`**: Python notebooks, web scrapers, and SQL data cleaning queries.
* **`/Data`**: Source datasets utilized across the projects.

---

## 📊 Featured Projects

### 1. AirBnB Full Project (Tableau)
* **Objective:** Analyzed market trends and pricing factors for AirBnB listings to optimize revenue.
* **Tools Used:** Tableau, Excel
* **Key Deliverable:** An interactive dashboard analyzing zip code metrics, revenue generation, and bedroom count impact.

### 2. Data Professional Survey Breakdown (Power BI)
* **Objective:** Cleaned and visualized global survey data to uncover salary trends and job satisfaction metrics in tech.
* **Tools Used:** Power BI, Power Query
* **Key Insights:** Broken down by favorite programming languages, work-life balance, and average income by role.

### 3. Bike Sales Performance Dashboard (Excel)
* **Objective:** Developed a dynamic dashboard for a retail bike company to track customer demographics and purchasing behavior.
* **Tools Used:** Excel (Pivot Tables, Advanced Formulas, Slicers)

### 4. Data Engineering & Automation Scripts (Python & SQL)
* **Amazon Web Scraper:** Python script using BeautifulSoup to automatically track product prices over time.
* **Automated File Sorter:** OS-level script to auto-organize complex file directories.
* **Crypto API Pipeline:** Real-time cryptocurrency data ingestion tool using automated API calls.
* **SQL Data Cleaning (Layoffs Dataset):** Thorough data cleaning pipeline handling duplicates, null values, and standardization in MySQL.
